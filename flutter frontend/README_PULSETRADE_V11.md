# PulseTrade V11 Ultimate Final

Premium Flutter finance dashboard with the original desktop/sidebar visual style preserved.

## Highlights
- Original premium PulseTrade UI kept as the default `flutter run` entry.
- Provider/services/screens folders preserved for scalable architecture.
- Dark/light mode and accent colors.
- Multi-currency trading display: EUR, USD, TRY, GBP.
- Dynamic stock detail pages for Siemens Energy, SAP, Tesla, NVIDIA and Apple.
- Separate chart datasets with 1D, 1W, 1M, 3M, 1Y, 5Y and Max range controls.
- Buy/Sell flow updates recent activity and portfolio state.
- AI Signal Center and professional controls added.
- News center, stock news cards, analysis, financials, dividends and profile tabs.
